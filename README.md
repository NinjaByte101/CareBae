CareBae is an AI-powered women's health chatbot that provides safe, private, and educational answers about menstrual health and wellness. It uses a RAG (Retrieval-Augmented Generation) pipeline to search through a curated library of verified medical documents like WHO Guidline, Research Papers and deliver accurate, context-aware responses — powered by Groq's LLaMA 3.1 LLM, Pinecone vector search, SentenceTransformers for embeddings, and Firebase for secure user authentication and conversation memory.
